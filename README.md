# 🚀 Projeto Integrador - Lógica e Algoritmos em Dart

Um repositório dedicado à exploração de conceitos fundamentais e lógica de programação utilizando a linguagem **Dart**. Este projeto reúne uma coleção de scripts focados na resolução de problemas práticos, operações matemáticas e manipulação de dados.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Dart

## ⚙️ Funcionalidades e Estrutura

O projeto é composto por diversos arquivos independentes, cada um com um objetivo específico:

* **`calculardesconto.dart`**: Algoritmo para o cálculo e aplicação de descontos.
* **`temperatura.dart`**: Conversor de unidades de temperatura.
* **`operacao.dart`**: Resolução de operações matemáticas.
* **`data.dart`**: Tratamento e manipulação de datas.
* **`frutas.dart`**: Estrutura de dados lidando com listas e arrays.
* **`gritar.dart`**: Manipulação de formatação de *strings*.
* **`saudacao.dart`** e **`alo.dart`**: Scripts base de interação de usuário e exibição de mensagens.

## 🚀 Como Executar

Para rodar qualquer um dos scripts localmente, é necessário ter o [Dart SDK](https://dart.dev/get-dart) instalado em sua máquina.

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/joaoerick1/Projeto-Integrador-.git](https://github.com/joaoerick1/Projeto-Integrador-.git)
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd Projeto-Integrador-
    ```
3.  Execute o arquivo desejado no terminal. Exemplo:
    ```bash
    dart run calculardesconto.dart
    ```
